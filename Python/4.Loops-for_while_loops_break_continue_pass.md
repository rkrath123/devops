break

```
for each in [3,4,8,9]:
    print(each)
    break
    
o/p-

------------------------------

for each in [3,4,8,9]:
    print(each)
    if each == 8 :
        break

continue

```
for each in [3,4,8,9]:
    if each == 4 :
        continue
    print(each)
